# 教訓ログ — 外国語動画 日本語字幕生成アプリ

## 形式

各教訓は以下の形式で記録する:

```
## [日付] カテゴリ: 教訓タイトル

**状況**: どんな問題・状況だったか
**判断ミス**: 何を間違えたか
**教訓**: 次回からどうするか
**ルール**: 自分への行動ルール
```

## ログ

## [2026-03-03] FFmpeg: ASS alignment 番号を仕様で確認すること

**状況**: `subtitle_position="top"` 時に alignment=6（中段右）を設定していた。
**判断ミス**: ASS 番号体系（7=左上, 8=中上, 9=右上 / 4=左中, 5=中中, 6=右中 / 1=左下, 2=中下, 3=右下）を誤って記憶していた。
**教訓**: FFmpeg/libass の alignment 値は必ず numpad 配置図で確認する。
**ルール**: 字幕 alignment を実装する際は `7 8 9 / 4 5 6 / 1 2 3` の numpad 配置を必ず参照し、top-center=8, bottom-center=2 を使う。

---

## [2026-03-03] FFmpeg subtitles フィルタ: パスエスケープはコロン・バックスラッシュ・シングルクォートの3種

**状況**: SRT パスのエスケープにコロンとバックスラッシュのみ対応し、シングルクォートが未対応だった。
**判断ミス**: `force_style='...'` をシングルクォートで囲んでいるため、パスにシングルクォートが含まれると FFmpeg フィルタが壊れる。
**教訓**: libass subtitles フィルタのパスエスケープ対象は `\`, `:`, `'` の3種。
**ルール**: `srt_escaped = path.replace("\\", "/").replace(":", "\\:").replace("'", "\\'")`
