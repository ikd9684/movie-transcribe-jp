# アプリ仕様書 — 外国語動画 日本語字幕生成アプリ

## 1. プロダクト概要

外国語で収録された動画をアップロードするだけで、AI による音声文字起こし・日本語翻訳を自動実行し、字幕付き動画と SRT ファイルをダウンロード提供する Web アプリ。

---

## 2. 機能要件

### 2.1 動画アップロード

| 項目 | 仕様 |
|------|------|
| 対応フォーマット | MP4, MOV, AVI, MKV, WebM |
| 最大ファイルサイズ | 2 GB |
| アップロード方式 | ドラッグ&ドロップ / ファイル選択ダイアログ |
| 複数ファイル | 非対応（1 本ずつ処理） |

### 2.2 音声文字起こし

| 項目 | 仕様 |
|------|------|
| エンジン | Whisper large-v3（ローカル実行） |
| 対応言語 | 英語・中国語・韓国語・スペイン語・フランス語・ドイツ語 など多言語 |
| 出力 | セグメント単位のテキスト + タイムスタンプ（開始/終了） |
| 言語自動検出 | あり（Whisper の detect_language 機能を使用） |

### 2.3 日本語翻訳

| 項目 | 仕様 |
|------|------|
| エンジン | Ollama (gpt-oss:20b) |
| 翻訳単位 | Whisper セグメント単位（前後 2 セグメントを文脈として付与） |
| 品質戦略 | システムプロンプトでニュアンス保持・自然な日本語を指示 |
| 並列処理 | 長時間動画では複数セグメントを並列翻訳 |

### 2.4 字幕生成・出力

| 項目 | 仕様 |
|------|------|
| SRT ファイル | タイムスタンプ + 日本語翻訳テキスト |
| 字幕焼き込み動画 | FFmpeg libass を使用して字幕をハードサブ |
| フォント | IPAexゴシック（日本語表示最適化） |
| 字幕スタイル | 下部中央配置、白文字・黒縁取り |

### 2.5 ダウンロード

- SRT ファイル単体ダウンロード
- 字幕焼き込み済み動画ダウンロード（元の動画と同一コーデック）

### 2.6 進捗表示

- SSE（Server-Sent Events）を使ったリアルタイム進捗通知
- 処理ステップごとの進捗率表示（0〜100%）
- エラー発生時は詳細メッセージを表示

---

## 3. 非機能要件

### 3.1 パフォーマンス

| 指標 | 目標値 |
|------|--------|
| アップロード速度 | 制限なし（クライアント帯域依存） |
| 文字起こし速度 | 実時間の 0.5 倍以下（GPU 使用時） |
| 翻訳速度 | セグメントあたり 2 秒以下（Ollama ローカル） |
| 字幕焼き込み | 実時間の 0.3 倍以下（CPU/GPU） |

### 3.2 可用性・信頼性

- ジョブはサーバー再起動後も状態を保持する（ファイルベースの状態管理）
- 処理失敗時はリトライなし。エラーメッセージを明示してユーザーに通知
- 同時処理ジョブ数: 1（リソース競合防止のためシリアル実行）

### 3.3 セキュリティ

- アップロードファイルの種別検証（MIMEタイプ + マジックバイト確認）
- アップロードファイルの保存先は専用ディレクトリのみ（パストラバーサル防止）
- ジョブ ID は UUID v4（推測困難）
- CORS はローカル開発用フロントエンドのオリジンのみ許可

### 3.4 互換性

- ブラウザ: Chrome 120+ / Firefox 121+ / Safari 17+
- OS: macOS（開発・実行環境）
- GPU: NVIDIA CUDA 対応（Whisper 高速化）または CPU フォールバック

## 4. 画面フロー

```
[TOP画面]
  ├── ヘッダー: アプリ名
  ├── VideoUploader: ドラッグ&ドロップエリア
  │     ├── ファイル未選択: "動画をドラッグ&ドロップ または クリックして選択"
  │     └── ファイル選択済み: ファイル名・サイズ表示 + "アップロード開始" ボタン
  │
  ├── [アップロード後] ProcessingStatus
  │     ├── ステップ表示:
  │     │     1. 音声抽出中... (FFmpeg)
  │     │     2. 文字起こし中... (Whisper)
  │     │     3. 翻訳中... (Ollama)
  │     │     4. 字幕生成中... (SRT作成)
  │     │     5. 字幕焼き込み中... (FFmpeg)
  │     ├── プログレスバー (0〜100%)
  │     └── エラー時: エラーメッセージ + "やり直す" ボタン
  │
  └── [処理完了後] ResultDownloader
        ├── "SRTファイルをダウンロード" ボタン
        ├── "字幕付き動画をダウンロード" ボタン
        └── "別の動画を処理する" ボタン
```

## 5. エラーハンドリング

| エラー種別 | 表示メッセージ | 対処 |
|-----------|--------------|------|
| 非対応フォーマット | 「対応フォーマット (MP4/MOV/AVI/MKV/WebM) のファイルを選択してください」 | アップロード拒否 |
| ファイルサイズ超過 | 「2GB 以下のファイルを選択してください」 | アップロード拒否 |
| Whisper エラー | 「音声の文字起こしに失敗しました。動画に音声トラックが含まれているか確認してください」 | ジョブ失敗 |
| Ollama 接続エラー | 「翻訳サービスに接続できませんでした。Ollama が起動しているか確認してください」 | ジョブ失敗 |
| FFmpeg エラー | 「動画処理に失敗しました。ファイルが破損していないか確認してください」 | ジョブ失敗 |
| ジョブ未発見 | 「ジョブが見つかりません。再度アップロードしてください」 | 404 返却 |
